Help & Documentation

This page explains the features of the Inflammation Prediction Tool, including prediction and exploratory data analysis (EDA).

Prediction

The prediction tool uses a Random Forest Regressor model to estimate inflammation marker values (such as IgA ASCA EU, IgG ASCA EU, OmpC. EU, Cbir1 EU, and ANCA EU) based on your uploaded microbiome and demographic data. A random forest is an ensemble machine learning method that builds many decision trees and averages their predictions for more accurate and robust results. When you upload your data and select a marker, the model analyzes the patterns in your features and generates a new column with predicted values for the selected inflammation marker.

Exploratory Data Analysis (EDA)

EDA helps you understand the relationships between your features and inflammation markers. When you upload a dataset containing both features and inflammation values, the tool generates correlation bar plots for each marker. These plots show how strongly each feature is related to a specific inflammation marker.

Understanding the Correlation Graphs

  • Positive correlation (bar to the right): As the feature increases, the inflammation marker tends to increase.
  • Negative correlation (bar to the left): As the feature increases, the inflammation marker tends to decrease.
  • The length of the bar shows the strength of the relationship. Longer bars mean a stronger correlation.
  • Only the top 20 most correlated features are shown for clarity.

Tips

  • For prediction, upload a CSV with all the feature columns used in model training. The tool will add a new column with predictions.
  • For EDA, upload a CSV with both features and inflammation marker values to generate the correlation plots.
